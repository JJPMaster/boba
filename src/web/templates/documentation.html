Welcome to Boba!

This is an AI platform for creating your own chatbots. All you need to make one is the following fields:
  • OpenAI API key
  • Conversation ID
  • Personal identity
  • Bot name
  • Message
How do these work? Well, they're each simple.

OpenAI API key

This field requires a valid key from an OpenAI paid account.

Conversation ID

This field allows you to choose a unique identifier for your conversation.

Personal identity

This is the prompt to be given to the AI. Preferably put it in the form "the ..."

Self-awareness suppression

This feature sends a request to the AI to essentially forget it's an AI for the time being. I recommend this feature be used always, except for if you like seeing "As an AI language model..."
